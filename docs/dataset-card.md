# PlantBrain AI — Dataset Card

## Why synthetic
Real plant records are confidential, so the demo uses a **curated synthetic corpus** that
mimics real industrial documents, plus a mechanism to fold in genuinely public documents.
Framing for judges: *"synthetic + public documents because real plant data is confidential;
the system is designed to plug into SharePoint, SAP PM, Maximo, and file servers."*

## Corpus (`data/synthetic/`, 53 documents, deterministic)
Plant **Shakti Petrochem Unit-2**; assets P-204A, HX-102, V-301, C-110, TK-501.

| Type | Count | Format |
|---|---:|---|
| OEM manuals | 5 | PDF |
| Work orders | 10 (+register) | PDF + CSV |
| Inspection reports | 8 | PDF |
| Incident reports | 6 | PDF |
| SOPs | 8 | PDF |
| Safety procedures | 5 | PDF |
| Compliance checklists | 5 | PDF |
| Sensor snapshots | 5 | CSV |

Generated by `scripts/generate_corpus.py` (reportlab, deterministic so hashes are stable).

## Storylines baked in (so agents have a real narrative)
- **P-204A** — repeated seal leakage + abnormal vibration; WO-141 closed **without** a
  post-repair vibration reading; INC-08 shows coupling misalignment as the recurring root
  cause. (Drives RCA + the multi-hop copilot demo.)
- **V-301** — hydrostatic pressure-test certificate overdue / not on file. (Drives compliance.)
- **HX-102** — tube-side fouling, rising outlet temperature. (Drives RCA #2.)

## Real public documents (optional enrichment)
`scripts/fetch_public_docs.py` registers real public docs (India Factories Act / OISD, CSB
incident reports) into `data/raw/real/`. Drop PDFs there and rerun to earn the eval's
"validated with real industrial document samples" credit. (Some gov sources bot-block direct
download; manual placement then re-run works.)

## Gold labels (`data/labeled/`)
`qa_gold.json`, `compliance_gold.json`, `entities_gold.json` — the evaluation ground truth.

## Provenance & integrity
Every chunk keeps its source document + page; every knowledge-graph edge keeps a source
chunk id. Documents are content-hashed (SHA-256) and de-duplicated on load.
