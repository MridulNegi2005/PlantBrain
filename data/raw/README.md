# data/raw

Public dataset placeholders. Large files are gitignored (`*.zip`, `*.tar.gz`, `*.csv`) — this README documents
where to fetch each one; download scripts land in Interval 1.

- `nasa_cmapss/` — NASA C-MAPSS turbofan degradation dataset (sensor telemetry, RCA garnish only).
- `ai4i/` — UCI AI4I 2020 Predictive Maintenance dataset.
- `mimii_optional/` — MIMII industrial machine-sound anomaly dataset (optional).
- `pid_samples/` — PID2Graph real-world P&ID dataset (pre-digitised offline, never live-parsed in the demo).
